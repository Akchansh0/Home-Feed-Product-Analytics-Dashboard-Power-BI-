# Home-Feed-Product-Analytics-Dashboard-Power-BI-
Home Feed Product Analytics Dashboard built in Power BI analyzing 10k+ users over 3 months. Tracks Active Users, Sessions, Engagement Rate, and Feed CTR across Android and iOS to identify engagement trends and optimize feed performance.

## Metrics
- Active Users
- Session Count
- Feed CTR
- Engagement Rate
- Avg Session Time

## Tools Used
- Power BI
- DAX
- Data Modeling

## Key Insights
- Engagement peaked in February
- CTR declined from Jan to Mar
- iOS users show slightly higher engagement
